# Simple Text Editor Using Tkinter

A lightweight text editor built with Python and Tkinter. The application supports opening, editing, and saving text files with keyboard shortcuts and a clean dark-themed interface.

## Features

- Open text files
- Edit and save files
- Save new files using "Save As"
- Keyboard shortcuts support:
  - Ctrl + S → Save
  - Ctrl + O → Open
  - Ctrl + C → Copy
  - Ctrl + V → Paste
  - Ctrl + X → Cut
- Dark-themed user interface
- Simple and beginner-friendly design

## Technologies Used

- Python 3
- Tkinter GUI Library

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/tkinter-text-editor.git
```

Move into the project directory:

```bash
cd tkinter-text-editor
```

## Usage

Run the application:

```bash
python main.py
```

## Keyboard Shortcuts

| Shortcut | Function |
|----------|----------|

| Ctrl + O | Open File |
| Ctrl + C | Copy Text |
